Website

Proyek dunia maya, dunia nyata yang lain

Disini aku sedang membuat website yang kamu lihat ini, aku membuat ini selama 4 hari. Ya, aku suka membuat website karena aku bisa mengeluarkan apa yang aku pikirkan tanpa perlu berbicara langsung.
Aku lupa sejak kapan aku belajar membuat website. Dulu aku pernah membuat website dengan PHP, tapi karena terlalu rumit jadi aku hanya menggunakan HTML, CSS dan JS. Aku menggunakan Visual Studio Code atau biasa disingkat VSC/VSCode, aku belajar membuat website sudah lama, mungkin sejak umur 7 tahun.

Aku mempelajari cara membuat website secara autodidak dan mencari tahu sendiri di Internet, aku berkali kali mencari solusi untuk masalah error, tapi jika itu terlalu rumit aku mencarinya bersama AI bernama Google Bard. Fasilitasku saat itu hanyalah komputer gaming keluaran lama, jadi aku senang bisa membuat kreasi tanpa harus menambah biaya peningkatan komputer.
Seberapa sulit aku mempelajarinya? Cukup sulit, hal paling sulit menurutku adalah, "mengingat berbagai macam dan jenis kode." Semua itu menyenangkan, tapi aku sering kahabisan ide untuk membuat website. Membuat website sangat boros ide, apalagi pemograman membuatku jadi mudah lupa.
